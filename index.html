CIENCIA DE DATOS

1. Webscraping

El web scraping es una técnica utilizada para extraer información de sitios web de manera automática. Es fundamental para recolectar datos masivos de diversas fuentes en línea.

Contexto del problema: Recolectar información de precios de productos en tiendas en línea para analizar tendencias de precios.

2. Técnicas de visualización

Las técnicas de visualización de datos permiten representar gráficamente la información para facilitar su interpretación. Esto incluye gráficos de barras, líneas, dispersión, entre otros.

Contexto del problema: Visualizar la relación entre el precio y la demanda de productos de un sitio de comercio electrónico.

Puedes acceder al hipervínculo del código en Google Colab.

3. Medidas de tendencia central

Las medidas de tendencia central son usadas para describir un conjunto de datos por su valor central o típico. Entre las más comunes están la media, la mediana y la moda.

Contexto del problema: Calcular la media de precios de diferentes productos electrónicos para determinar un precio representativo.

Puedes acceder al hipervínculo del código en Google Colab.

4. Medidas de dispersión

Las medidas de dispersión permiten entender la variabilidad o dispersión de un conjunto de datos. Algunas de estas medidas incluyen la desviación estándar y el rango intercuartílico.

Contexto del problema: Evaluar la dispersión de los precios de los productos en diferentes regiones del país.

Puedes acceder al hipervínculo del código en Google Colab.

5. Conclusiones

La ciencia de datos permite resolver problemas complejos al analizar grandes volúmenes de información. En este proyecto, hemos aplicado diferentes técnicas para recolectar, visualizar y analizar datos.

Conclusión principal: La integración de técnicas como el web scraping, la visualización de datos y las medidas estadísticas es crucial para entender tendencias y comportamientos en grandes conjuntos de datos.
